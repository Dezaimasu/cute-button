▶ Show/hide rules All specified paths are relative to your default save folder. If you leave it empty, it will save files in default save folder. if you leave "filename" field empty, files will be saved with default filename. File extension will be set automatically, please don't specify it explicitly. Using placeholders from the list below you can save files from each domain in their own folders, or with some specific filenames. Feel free to request more List of available placeholders, must be surrounded with :: ::domain:: Domain name of the current site ::title:: Title of the current page ::board_name:: Board name if exists (without // symbols) ::thread_number:: Thread number if exists ::date:: Current date ::time:: UNIX timestamp ::filename:: Default filename as assigned by browser ::original_filename:: Original filename, available on most popular imageboards You can choose mouse button or keyboard hotkey or both for each template. You can specify domain name if you want some hotkey/save template to work only on this specific site. You can also exclude hotkey for specific site, so it will work everywhere except this specific site. To do this add domain name starting with "-" symbol. Domain name you need to specify is usually shown in address bar, or you can open Web Console (Ctrl+Shift+K) and write document.location.hostname, the result will be domain name you can insert into "host" field. Rules with "host" have higher priority than "-host" rules. Rules without host specified have lowest priority. Examples: wikipedia.org : this hotkey will only work on wikipedia.org -bing.com : this hotkey will work everywhere except bing.com
