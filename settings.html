Save path MUST be relative to your downloads folder (you can check it on Tools -> Options -> General -> Downloads -> Save files to).
E.g. if your downloads folder is "D:\Pictures", and you want this extension to save files in "D:\Pictures\happy-cute-time\dank-memes",
you just need to specify "happy-cute-time\dank-memes".

Unfortunately, extensions can't save files outside of your downloads folder ("D:\Pictures" in your case).
If you REALLY want to do that, look for "symbolic links".

Also make sure your hotkeys won't overlap with browser hotkeys and with each other.
